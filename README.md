# Julians-durag-raindow!

## A single-page application dedicated to helping Julian figure out what durags to buy to finish out his collection. You will find out which ones he already owns buy moving your mouse over the buttons with labeled colors on them. Then add new ones by googling them, copying the image address and putting it in the lines above! Once you open the index.html file everything will be laid out for you. Go ahead and fork this repository. then copy the code with an SSH key.

## Getting Started

# This a custom API so you will run this locally.

* Clone the repo with the SSH key selected with the command 'git clone git@github.com:Brick5g/Julians-durag-raindow-project.git'

* Run: npm install -g json-server

* Use command json-server --watch public/db.json to start the server

## Instructions

* In an new terminal run the command open index.html to open the website

* Read the text on the screen to see the features and objective

* On the right side, mouseover the colors to see what color durags Julian owns already.

* Have fun helping him find what durags to buy next!
